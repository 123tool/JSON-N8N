# ⚡ WhatsApp AI Business Agent - Pro Edition (n8n)

Workflow n8n tingkat tinggi yang mengubah WhatsApp Business menjadi mesin sales otomatis dengan otak GPT-4o.

## 🔥 Apa yang Baru di V2.0?
- **Logic Sapaan Pintar:** Tidak lagi spam "Halo Kak" di setiap balasan.
- **Deep Sheets Integration:** AI bisa membedakan stok ready, menipis, atau kosong secara real-time.
- **Anti-Markdown Formatting:** Output dioptimalkan khusus untuk tampilan layar WhatsApp agar link tidak rusak.
- **Sales Persuasion:** AI tidak hanya menjawab, tapi mendorong closing/transaksi.

## 🛠️ Cara Pakai
1. Import `workflow.json` ke n8n.
2. Set **System Message** menggunakan prompt yang ada di folder `/prompts`.
3. Hubungkan Credentials (WA Cloud API, OpenAI, Google Sheets).
4. Jalankan!

## 📦 Folder Structure
- `/workflow`: File JSON n8n.
- `/prompts`: Instruksi cerdas (System Message).
- `/docs`: Panduan setup API Key.
